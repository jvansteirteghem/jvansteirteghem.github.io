Twunnel: A HTTP/SOCKS5 Tunnel for Twisted

Aug 22nd, 2013

Twunnel is a HTTP/SOCKS5 tunnel for Twisted.

Twunnel supports:

  • HTTP
  • HTTP + Basic authentication
  • SOCKS5

The Twunnel project page: https://github.com/jvansteirteghem/twunnel

JAP 3.0.0 Released!

Aug 8th, 2013

JAP is a SOCKS5 proxy server written in PYTHON and JAVASCRIPT. JAP has a local proxy server (which supports TCP, TCP over SSH and TCP over WS/WSS) and a remote proxy server (which supports TCP over SSH and TCP over WS/WSS).

  • JAP_LOCAL_PYTHON
  • JAP_LOCAL_PYTHON_WINDOWS
  • JAP_LOCAL_SSH_PYTHON
  • JAP_LOCAL_SSH_PYTHON_WINDOWS
  • JAP_LOCAL_WS_NODE
  • JAP_LOCAL_WS_PYTHON
  • JAP_LOCAL_WS_PYTHON_WINDOWS
  • JAP_REMOTE_SSH_PYTHON
  • JAP_REMOTE_SSH_PYTHON_WINDOWS
  • JAP_REMOTE_WS_NODE
  • JAP_REMOTE_WS_NODE_MODULUS
  • JAP_REMOTE_WS_NODE_NODEJITSU
  • JAP_REMOTE_WS_NODE_OPENSHIFT
  • JAP_REMOTE_WS_PYTHON
  • JAP_REMOTE_WS_PYTHON_OPENSHIFT
  • JAP_REMOTE_WS_PYTHON_WINDOWS

JAP also has a UI.

  • JAP_PYTHON
  • JAP_PYTHON_WINDOWS
