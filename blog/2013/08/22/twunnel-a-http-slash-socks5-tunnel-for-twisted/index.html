Twunnel: A HTTP/SOCKS5 Tunnel for Twisted

Aug 22nd, 2013

Twunnel is a HTTP/SOCKS5 tunnel for Twisted.

Twunnel supports:

  • HTTP
  • HTTP + Basic authentication
  • SOCKS5

The Twunnel project page: https://github.com/jvansteirteghem/twunnel
